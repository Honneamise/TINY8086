DWORD            Data Type for 4 bytes                              Data Type
 
    Used to assign the data type of a variable as a double-word (4 bytes).
 
       Notes:     This data type is used by the:
 
                        *    EXTRN directive
                        *    PTR operator
                        *    LABEL directive
                        *    THIS operator
 
                  Use DD to define a variable of type DWORD.
