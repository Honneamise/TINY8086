INT 22h (34)             Terminate Address
 
    Contains the address to which a program transfers control upon
    termination. When a program is executed, DOS copies this address into
    offset 0Ah through ODh of the Program Segment Prefix.  DOS normally
    sets this address when it executes a program with the EXEC function
    call.
