INT 00h (0)              Divide by 0
 
    An INT 0 is generated whenever a division-by-zero error condition
    occurs.
 
    INT 0 is generated by IDIV or DIV when the result won't fit into the
    destination, or the divisor is 0.
 
    The default handler for this interrupt will display the message
    'Divide by Zero', then return control to DOS.
 
       Notes:         Care must be taken when writing a division-by-zero
                      interrupt handler. The 8088, 8086, V20, and V30 push
                      the CS:IP of the instruction FOLLOWING the DIV or
                      IDIV instruction that generated the division-by-zero
                      error condition. The 80286 and 80386, however, push
                      the CS:IP of the DIV or IDIV instruction.
