WORD             Data Type for 2 bytes                              Data Type
 
    Used to assign the data type of a variable as a word (2 bytes).
 
       Notes:     This data type is used by the:
 
                      *    EXTRN directive
                      *    PTR operator
                      *    LABEL directive
                      *    THIS operator
 
                  DW is used to define a variable of type WORD.
