JGE              Jump If Greater or EQual            Flags: not altered
 
JGE short-label
 
          Jump Condition:  Jump if SF = OF
 
    Used after a CMP or SUB instruction, JGE transfers control to short-
    label if the first operand is greater than or equal to the second.
    (Both operands are treated as signed numbers.)  The target of the jump
    must be within -128 to +127 bytes of the next instruction.
 
  --------------------------------------------------------------------------
   Operands                  Clocks   Transfers  Bytes   Example
   short-label              16 or 4       -        2     JGE GREATER_EQUAL
  --------------------------------------------------------------------------
 
       Notes:         JNL, Jump if Not Less, is the same instruction as
                      JGE.
 
                      JAE, Jump if Above or Equal, should be used when
                      comparing unsigned numbers.
 
                      JGE, Jump if Greater or Equal, should be used when
                      comparing signed numbers.
