0:465h                   Current Setting of the Mode Control Register
 
 Length: 1 byte
 
    Contains the current setting of the Mode Control register (port 3x8h).
 
  --------------------------------------------------------------------------
 
    Depending upon which CRT controller is currently active, this value
    could reflect either the contents of:
 
                      3B8h for Monochrome, or
                      3D8h for Color Graphics Adapter
